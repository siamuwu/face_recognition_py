# Face Recognition Project by Python
### Used: OpenCV, Pickle and Numpy

I have used openCV for this project. It only works when the user places their front face into the camera, the system then recognizes the user by comparing it with the faces that is given in the face recognition folder. If you want to use it:
* You have to store the images with their name in a folder which should be under same folder where the face recognition related codes exist.
* First you have to run the encode_known_image file then face_recognition file, it actually reads the faces to compare it with the user one.
* The photos you want to save as reference must be front facing, and be clicked in a well lit room.
* Use your webcam to detect yourself after running face_recognition file.
