# Face Recognition Project by Python
### Used: OpenCV, Pickle and Numpy

I have used OpenCV for this project. It works when the user places their face in front of the camera then the system recognizes the user by comparing it with the faces stored in the face recognition folder. If you want to use it:
* You have to store the images with their names in a folder, which should be in the same directory where the face recognition codes exist.
* Firstly you have to run the encode_known_image file, then the face_recognition file. It reads the stored faces and compares them with the user’s face.
* The photos you want to save as references must be front-facing and taken in a well-lit environment.
* Use your webcam to detect yourself after running the face_recognition file.
